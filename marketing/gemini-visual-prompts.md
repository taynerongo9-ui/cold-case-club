# Cold Case Club — Gemini Image Generation Prompts

## Brand Visual Guidelines

Before using these prompts, understand the visual language:
- **Color palette:** Deep noir blacks (#0a0a0f), warm golds (#c9a84c), blood reds (#8b2020), aged cream/parchment
- **Mood:** Dark, atmospheric, cinematic — think David Fincher meets vintage detective noir
- **Lighting:** Low-key, dramatic shadows, single warm light sources (desk lamps, candles, window light)
- **Textures:** Aged paper, coffee stains, typewriter text, worn leather, brass fittings
- **Era feel:** 1940s-1990s detective aesthetic, never modern or digital

## Production Notes

### Gemini Prompting Best Practices Used:
1. **Specificity over ambiguity** — every prompt includes exact materials, lighting direction, color palette, and mood
2. **Reference anchoring** — "think David Fincher meets..." gives Gemini a style reference point
3. **Negative constraints** — "No digital elements. No screens." prevents unwanted modern objects
4. **Technical specifications** — aspect ratio, resolution, photography style explicitly stated
5. **Emotional descriptors** — mood words (ominous, inviting, melancholy) guide the overall feel
6. **Composition guidance** — "focal point," "shallow depth of field," "shot from above" direct framing
7. **Context-aware detail** — period-appropriate cars, era-correct architecture, historically accurate materials

### Image Sizing Guide:
| Use Case | Dimensions | Ratio |
|----------|-----------|-------|
| Hero banner | 1920x1080 | 16:9 |
| OG/Social share | 1200x630 | ~1.9:1 |
| Instagram feed | 1080x1080 | 1:1 |
| Instagram Story | 1080x1920 | 9:16 |
| Pinterest pin | 1000x1500 | 2:3 |
| Blog header | 1200x675 | 16:9 |
| Product card | 800x1000 | 4:5 |

### Post-Generation Checklist:
- [ ] Does the image match our dark noir aesthetic?
- [ ] Are gold (#c9a84c) and deep black (#0a0a0f) the dominant tones?
- [ ] Does it feel premium ($75 product, not $5)?
- [ ] Is there space for text overlay where needed?
- [ ] No anachronistic elements (smartphones, modern tech) in period shots?
- [ ] Would this stop someone scrolling on Instagram?
- [ ] Does it tell a story in a single frame?

### Recommended Generation Settings:
- Use Gemini 2.0 Flash or Gemini 2.5 Pro for best photorealistic output
- Generate 4 variations per prompt, select the strongest
- Request "photorealistic" and "commercial photography quality" in every prompt
- For consistency across images, include the phrase "dark noir color palette with warm gold accents, deep blacks, and muted warm tones" in every prompt
